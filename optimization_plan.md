# 게임 성능 최적화 및 리팩토링 계획

## 🚀 즉시 적용 항목
1. **Canvas 렌더링 최적화**
   - 배치 렌더링 구현 (드로우 콜 70% 감소)
   - 프레임 당 렌더링 객체 수 제한
   - 오프스크린 캔버스 활용

2. **자원 관리 시스템**
   - 이미지 스프라이트 아틀라스 자동화 (public/game_images/* 통합)
   - 텍스처 LOD 적용 (거리별 디테일 조정)
   - WebGL 텍스처 압축

3. **코어 게임 루프 개선**
   - requestAnimationFrame 기반 틱 시스템
   - 물리/로직 업데이트 분리
   - Web Worker를 이용한 타격 계산 오프로딩

## ⚙️ 중기적 개선 항목
1. **메모리 관리 강화**
   - 객체 풀링 패턴 도입 (GC 발생 감소)
   - ArrayBuffer 활용 데이터 버퍼링

2. **아키텍처 개편**
   - ECS (Entity-Component-System) 구조 적용
   - 이벤트 버스 패턴 도입

3. **타입 안정성 강화**
   - TypeScript 마이그레이션 로드맵
   - 핵심 모듈부터 점진적 적용

## 📊 성능 측정 지표
- FPS 평균 ≥ 60 유지
- 메모리 사용량 30% 감소
- 첫 로딩 시간 50% 단축
